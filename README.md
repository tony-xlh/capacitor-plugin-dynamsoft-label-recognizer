# capacitor-plugin-dynamsoft-label-recognizer

Capacitor plugin of Dynamsoft Label Recognizer which brings text recognition ability to your apps.

## Install

```bash
npm install capacitor-plugin-dynamsoft-label-recognizer
npx cap sync
```
